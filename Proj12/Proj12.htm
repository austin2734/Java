PROGRAM SPECIFICATIONS

Begin with the files that you downloaded named Proj12.java and Proj12Skeleton.txt.

Rename the text file from Proj12Skeleton.txt to Proj12Runner.java. At that point the program will compile and run but it will not meet the specifications given below.

You must update the file named Proj12Runner.java to cause the program to meet the specifications by:

  1. Inserting your name where indicated.
  2. Modifying only one other physical line of code.

Note that you must not modify code in the file named Proj12.java.
